# RiverTubing-Web-Application

Source code website untuk layanan **River Tubing** yang menyediakan informasi wisata serta fitur interaksi pengguna seperti kritik & saran dan blog post.



## Deskripsi
![1736953505169](https://github.com/user-attachments/assets/3f7389aa-fb72-4597-bf7c-faa31974dfda)
**RiverTubing-Web-Application** adalah website informatif dan interaktif yang dirancang untuk memberikan informasi seputar kegiatan river tubing serta menjadi wadah bagi pengguna untuk berbagi pengalaman dan memberikan masukan.

Website ini dapat digunakan oleh pengelola wisata river tubing untuk:
- Menyampaikan informasi terbaru
- Membagikan artikel atau blog post
- Menerima kritik dan saran dari pengunjung

---

## Fitur Utama

### User Features
- Mengirim kritik dan saran
- Melihat informasi seputar river tubing
- Membaca blog post

### Admin Features
- Membuat blog post
- Mengedit dan menghapus blog post
- Melihat kritik dan saran dari pengguna
- Mengelola konten informasi river tubing

---

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP / Laravel 
- **Database**: SQLite
- **Framework / Library**: Bootstrap
